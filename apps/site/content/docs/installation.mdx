## Manual Installation

If you prefer manual setup or need more control, follow the steps below:

<Timeline>
## Install Dependencies

```bash
npm install ui-lab-components
# or
pnpm add ui-lab-components
```

## Import Styles

Add the component styles to your application root:

```tsx
// app/layout.tsx (Next.js) or main.tsx (Vite)
import 'ui-lab-components/styles.css';
```

## Configure Tailwind CSS v4

Ensure your project uses Tailwind CSS v4 with PostCSS:

```javascript
// postcss.config.mjs
export default {
  plugins: {
    '@tailwindcss/postcss': {},
  },
};
```

## Start Using Components

```tsx
import { Button, Card, Input } from 'ui-lab-components';

export default function Home() {
  return (
    <Card>
      <Card.Header>
        <Card.Title>Welcome to UI Lab</Card.Title>
      </Card.Header>
      <Card.Content>
        <Input placeholder="Your email" />
      </Card.Content>
      <Card.Footer>
        <Button>Get Started</Button>
      </Card.Footer>
    </Card>
  );
}
```
</Timeline>
