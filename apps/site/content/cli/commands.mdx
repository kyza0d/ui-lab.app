# CLI Commands Reference

Complete reference guide for all available UI Lab CLI commands.

## Component Commands

### ui-lab add

Add a component to your project:

```bash
ui-lab add button
```

### ui-lab list

List all available components:

```bash
ui-lab list
```

## Configuration Commands

### ui-lab config

View or modify configuration:

```bash
ui-lab config --show
```

## Documentation Commands

### ui-lab docs

Generate documentation:

```bash
ui-lab docs --output ./docs
```
