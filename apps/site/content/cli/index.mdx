# CLI Documentation

The UI Lab command-line interface provides powerful tools for managing components and configurations from the terminal.

## Overview

The UI Lab CLI enables you to:

- Install and manage components
- Create new component scaffolds
- Configure design tokens
- Generate documentation
- Optimize build output

## Getting Started

This section covers all CLI commands and their usage patterns to help you work efficiently with UI Lab from the command line.
