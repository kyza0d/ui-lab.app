# Getting Started

## Installation

Install UI Lab components in your project:

```bash
npm install @ui-lab/react
```

## Basic Usage

Import and use components in your code:

```jsx
import { Button, Input, Select } from '@ui-lab/react';

export function MyForm() {
  const [email, setEmail] = useState('');

  return (
    <div>
      <Input
        type="email"
        value={email}
        onChange={(e) => setEmail(e.target.value)}
        placeholder="Enter your email"
      />
      <Button variant="primary">Submit</Button>
    </div>
  );
}
```

## Using with Claude

### Enable the MCP Server

To use UI Lab components with Claude, enable the MCP server in your Claude settings:

1. Go to Claude settings
2. Look for MCP Servers
3. Add `@ui-lab/mcp`
4. Claude will now have access to live component information

### Asking Claude to Generate Components

You can now ask Claude to generate UI code:

```
Generate a user login form with email and password fields,
plus a submit button. Use UI Lab components.
```

Claude will generate type-safe, accessible component code using the UI Lab registry.

## Component Variants

Most components support variants for different use cases:

```jsx
// Button variants
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="outline">Outline</Button>
<Button variant="ghost">Ghost</Button>

// Sizes
<Button size="sm">Small</Button>
<Button size="md">Medium</Button>
<Button size="lg">Large</Button>
```

## Theming

Components automatically respect your theme configuration:

```jsx
<ThemeProvider theme="light">
  <App />
</ThemeProvider>
```

Themes use semantic design tokens for consistent styling across all components.

## Accessibility

All UI Lab components include accessibility features by default:

- Proper ARIA labels and roles
- Full keyboard navigation
- Focus indicators
- Color contrast compliance

No additional configuration needed.
